# CPM Package Repository 🚀

A fast, minimal and human-friendly XBPS package repository for CPM.

CPM is a simple wrapper around XBPS designed for Android proot environments and lightweight Linux systems.

---

## 📦 What is this repo?

This repository contains:
- Prebuilt packages for Android-sdx environments
- Dev tools (clang, cmake, node, python, etc.)
- Lightweight utilities
- Custom CPM packages
- Future experimental builds

---

## ⚡ Why CPM Repo?

Because mobile dev shouldn't feel like ancient ritual scripting.

CPM + this repo gives you:
- Simple install syntax
- Fast package access
- Android-friendly tooling
- Proot-ready binaries
- Minimal setup pain
